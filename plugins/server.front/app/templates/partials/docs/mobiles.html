The SDKs for mobiles

If you are doing a mobile application, you should use an OAuth.io's SDK according to your language. We currently support Android, iOS, Apache Flex, and Apache Cordova / PhoneGap.

All the features from the javascript web SDK are present, but there are some differences in the coding style depending on the language. To see the specific documentation for an SDK, you can visit the SDK's page on github:
