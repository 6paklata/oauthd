Installation & initialization

Download OAuth.js (2.2kb)
  • Download OAuth.js - minified (1.4k)

Download the Javascript SDK and, in your HTML, load the SDK using a <script> tag.
You can use cdnjs to load fastly the lastest version of the SDK.

<script type="text/javascript" src="/path/to/oauth.js">

Once the SDK loaded, you have to initialize it in your Javascript code. It's used to configure the SDK with your OAuth.io public key

Basic usage

You have 2 ways of authorizing your user:

OAuth.initialize('{{app.key||"Your_Public_Key"}}')
  • Open a popup with the {{provider}} authorize form
OAuth.popup('{{provider}}', function(err, result) {
  // handle error with err
  // use result to access tokens or make API calls
});

OR

  • Redirect the user to the {{provider}} authorize form
OAuth.redirect('{{provider}}', "callback/url");

In the redirect mode, you have to catch the response in your callback page using OAuth.callback

OAuth.callback('{{provider}}', function(err, result) {
  // handle error with err
  // use result to access tokens or make API calls
});

Make an authorized API call

Once your user authorized, in the callback function, you can easily make API calls without having to know what is an access_token. We've put in the SDK some methods to make your API call as simple as it should be.

You can use get(), post(), put(), delete() and patch() to make your API calls. As the tokens are in the result of the authorization, we totaly abstract them by adding some high-level methods.
